# Full Stack Item List App

## Description

A full-stack application built with React, Node.js, Express, and SQLite.
The project demonstrates core full-stack concepts including REST APIs, state management and persistent data storage.

Users can add and view items stored in a SQLite database.

## Features

- View a list of items fetched from a backend API
- Add new items via a React frontend
- Persistent data storage using SQLite
- RESTful API built with Express
- Cross-origin communication using CORS

## Technologies Used

- React
- Node.js
- Express
- SQLite
- CORS

## How to Run

## Backend

```bash
cd Back-end
npm install
node index.js
```

## Frontend
```bash
cd Front-end
npm install
npm start
```
